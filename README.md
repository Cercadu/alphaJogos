# AlphaDino 3D Platformer

Plataforma de jogos retrô-cibernética em 3D rodando no navegador (WebGL/Three.js) com suporte completo a dispositivos móveis (controles de joystick flutuantes e feedback háptico).

## 🎮 Funcionalidades
- **Gráficos 3D Voxel:** Todo o jogo renderizado com blocos 3D realistas, sombras e névoa.
- **5 Fases Únicas:** Diferentes cenários (Deserto, Cavernas, Cidade e Castelo de Lava) e mecânica clássica de mastro de bandeira no fim do nível.
- **5 Dinossauros Jogáveis:** Cada um com suas próprias habilidades (atirar fogo, planar, escudos, dash e ímã de moedas).
- **Controles Mobile Em FIFA Mobile Style:** Joystick flutuante na esquerda para andar e botões de ação na direita.
- **Service Worker (PWA):** Instale direto no celular e jogue offline sem internet.
- **Retro Synth Sounds:** Efeitos sonoros sintetizados na hora usando a Web Audio API.

## 🚀 Como Executar Localmente
Apenas inicie um servidor HTTP estático na raiz do projeto:
```bash
# Com python
python -m http.server 8080
```
Abra `http://localhost:8080` no navegador.
